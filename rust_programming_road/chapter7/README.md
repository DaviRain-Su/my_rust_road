Q6: 关于命令行字体颜色问题：
A6： echo "\e[30mHello\e[0m", 这是设置的字体颜色为黑色，所以30是字体颜色黑色
30-37字体颜色(黑色， 红色， 绿色，黄色，蓝色，粉色，靛蓝色， 白色)
40-47是设置背颜色的也是从黑色到白色
echo "\e[37;40mHello\e[0m"
echo "\e[40;37mHello\e[0m",所以这两个的颜色设置是一样的，都是字体颜色为白色，背景颜色为黑色。
